This project applies Topic Modeling on SPARQL query results. 
To recognize if one dataset contains the same topics as another.

#Requirements------------------------------------------------------------------

Have lates version of R and Rstudio.
Download the following packages: SPARQL, mallet, rjava, ggplot2

#Procedure------------------------------------------------------------------------------

"Main" contains all scrips needed.
You need to run them in this given order:
1-Functions
2-SPARQL
3-TopicModeling
4-Visualiztion
